# 🎓 Teachio - Home Tuition Marketplace

Teachio is a modern home tuition booking platform that connects students and parents with verified tutors in their local area. It allows tutors to create profiles and offer home tuition services, while parents can search, compare, and book tutors easily.

---

## 🚀 Features

- 🔐 Google Authentication (Supabase)
- 👨‍🏫 Tutor profile creation
- 🔍 Search tutors by subject, class, and location
- 📍 Location-based tutor discovery
- ⭐ Ratings and reviews system
- 💬 Direct contact between parents and tutors
- 📅 Booking / tuition request system
- 🛡️ Verified tutor system (trust & safety)

---

## 💡 Project Vision

Teachio aims to become the **"OYO for Education in Nepal"**, creating a trusted network of home tutors where quality education is accessible, safe, and easy to book.

---

## 🧱 Tech Stack

- Frontend: React (Vite)
- Backend: Supabase
- Authentication: Supabase Auth (Google OAuth)
- Database: PostgreSQL (Supabase)
- Hosting: Vercel / Netlify
